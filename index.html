HTML (Hyper Text Markup Language)

HTML is the standard markup language for creating web pages.It describes the structure of a web page using markup.

  • Block Elements: <h1>, <p>, <ul>, <ol>
  • Inline Elements: <i>, <strong>, <image>
  • Attributes: type, src, alt, href
Tag Description
<h1> Main Heading
<p> Paragraph
<ul> Unordered List
<ol> Ordered List
<div> Generic Container
<hr> Horizontal Rule
<header> Header of a section or a page
<main> main content of a document
<li> List Item
<nav> Navigation links
<form> HTML form
<footer> footer of a section or a page
<table> Table
<article> article content
<a> Anchor
<i> Italic
<b> Bold
<br> Break
<strong> Strong/Bold
<input> Input
<img> image
<sub> Subscripted
<sup> Superscripted
<span> Span
type type is attribute written in starting tag
src attribute written in starting tag to define source of required link.
alt Alternate
href Hypertext Reference

CSS (Cascading Style Sheets)

CSS is used for styling web pages.It describes how elements should be displayed on the screen.

  • Font Properties: font-family, font-size, color
  • Box Model: margin, padding, border
  • Text Properties: text-align, line-height, text-decoration
Property Description
font-family Defines the font of the text
color sets the text color
font-size Adjust font size
font-style The font style property sets the style of the font.
font-weight The font-weight property controls the thickness or boldness of the text.

JavaScript

JavaScript is a programming language that enables interactive web pages.It can dynamically modify content.

  • Variables and Data Types
  • Functions and Control Structures
  • DOM Manipulation
Property Description
Content The content area is defined by the width and height properties.
Padding The space between the actual content and the border of the element is the padding.
Border The border surrounds the content and padding and gives the visual border of the element.
Margin The margin is the space outside the element that separates it from other elements in the layout.
